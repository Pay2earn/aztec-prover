## ⚡ Quick Start

* Clone repo
```bash
git clone https://github.com/Pay2earn/Aztec-Prover-Node.git
cd Aztec-Prover-Node
```
* Give permission to run the script
```bash
chmod +x aztec.sh
```

* Open a screen to run it in background
```bash
screen -S aztec
```

* Run with sudo
```bash
sudo ./aztec.sh
```
